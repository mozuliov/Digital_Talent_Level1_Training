# Digital Talent — Level 1

A 12-session, 120-minute-per-session training programme designed to transition financial analysts from **"Guardians"** of data into **"Catalysts"** of growth — equipping them with the digital and AI skills needed to build decision-support tools, automate routine workflows, and operate confidently in an AI-augmented finance function.

> **Note:** This repository does not contain the complete training materials. It is maintained specifically to share practice notebooks, templates, recommended reading lists, and supplementary resources that participants need during and after each session.

---

## Programme Structure

### Month 1 — The Transition to Digital Producer

| # | Session | Description |
|---|---------|-------------|
| 01 | **The Catalyst Identity** | Establishes a digital-first professional identity through the "Start, Stop, Continue" change framework, helping analysts identify high-ROI automation targets in their current workflows. |
| 02 | **Python for Financial Statistics** | Introduces Python and Jupyter Notebooks through hands-on regression analysis exercises, covering core libraries such as NumPy, pandas, statsmodels, and matplotlib. |
| 03 | **Automated Data Retrieval** | Covers data profiling, SQL, and Python-based ETL extraction techniques, enabling analysts to retrieve and automate data pipelines from various sources efficiently. |
| 04 | **Data Wrangling & Quality Management** | Teaches data cleaning, transformation, and quality assurance practices using Python (pandas, NumPy) to prepare datasets for advanced modelling and AI workflows. |

### Month 2 — Advanced AI & Decision Support

| # | Session | Description |
|---|---------|-------------|
| 05 | **Machine Learning for Predictive Forecasting** | Provides a conceptual and practical introduction to supervised machine learning, applying decision tree models to structured financial data for forecasting and classification. |
| 06 | **Scenario Modeling & Probabilistic Thinking** | Explores Monte Carlo simulation techniques to model uncertainty, quantify risk, and build richer, probabilistic views of financial scenarios. |
| 07 | **GenAI Code Assistance (Vibe Coding)** | Demonstrates how to leverage generative AI tools (GitHub Copilot, ChatGPT, Claude) to rapidly prototype and write code, lowering the technical barrier for financial analysts. |
| 08 | **Managing Agentic AI Workflows** | Introduces agentic AI concepts and orchestration frameworks, showing how to design and oversee multi-step automated workflows that can handle judgment-based finance tasks. |

### Month 3 — Strategy, Risk & Agile Delivery

| # | Session | Description |
|---|---------|-------------|
| 09 | **Strategic Data Storytelling & Visuals** | Covers principles of data narrative and visual communication, with a hands-on workshop building interactive Power BI dashboards to influence stakeholder decisions. |
| 10 | **AI Risk, Bias & Governance** | Examines algorithmic bias, model hallucinations, and data privacy considerations, with practical exercises on testing models for ethical risk and building accountability frameworks. |
| 11 | **Agile Product Management for Finance** | Applies agile "test-and-learn" methodologies to finance work, teaching participants to organise deliverables into sprints and manage a continuous finance-product backlog. |
| 12 | **Future Roadmap & Continuous Evolution** | Closes the programme with a forward-looking lecture on the future of finance and a personal development planning workshop to sustain digital growth beyond the course. |

## Repository Contents

Each session folder may contain a subset of the following:

- **`A ...`** — Lecture notes and reading materials
- **`B ...`** — Hands-on Jupyter Notebook exercises
- **`C ...`** — Charts, infographics, and visual templates
- **`D ...`** — Supplementary and recommended reading lists

---

## Getting Started

Most practical exercises are designed to run in **Google Colab** — no local setup required. Simply open the relevant notebook link provided in each session folder.

For local execution, install the dependencies listed in the `requirements.txt` file inside each session folder:

```bash
pip install -r requirements.txt
```
